# 🐶 TinDog – A Tinder for Dogs

A fun and responsive landing page for **TinDog**, a fictional dating platform that helps dogs find their perfect match. Built using **Bootstrap 5**, this project demonstrates how to create a modern, mobile-first landing page with reusable components and responsive layouts.

## 📖 About

TinDog is a playful take on the popular dating app concept—but for dogs! This project features a complete landing page with a hero section, feature highlights, testimonials, pricing plans, and a responsive footer, all built using Bootstrap's powerful grid system and components.

## 🚀 Features

* 🐶 Eye-catching hero section
* 📱 Fully responsive Bootstrap layout
* ✨ Gradient background with CSS animation
* ⭐ Feature cards with Bootstrap Icons
* 💬 Customer testimonial section
* 💳 Responsive pricing cards
* 📄 Multi-column footer
* 🎨 Modern and clean UI
